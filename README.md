# Книжный Справочник
 Сервис по получению информации о книгах и авторах

## Описание сущностей
Лицензия (licence) , которой может владеть только один автор:
>* id
>* number (номер лицензии)

Автор (author), который владеет только одной лицензией и многими книгами:
>* id
>* name
>* List Book
>* licence

Книга(book):
>* id
>* name
>* author
